## 系統概述

本系統是一個 AI 資安資料集審核平台，主要功能包括：
- 用戶認證與權限管理
- 資料審核流程
- 統計儀表板
- 管理員功能（資料管理、系統設定、用戶管理等）
- 響應式設計支援桌面和手機版

### 用戶角色
- **admin**: 系統管理員，擁有所有功能權限
- **expert**: 資安專家，主要進行資料審核

## 測試環境準備

### 前置條件
1. 確保後端服務正常運行（FastAPI）
2. 確保前端服務正常運行（Vue.js）
3. 準備測試用的用戶帳號：
   - admin/admin
   - expert/expert
4. 準備測試資料（法規條文、待審核資料等）

### 測試工具建議
- 瀏覽器開發者工具
- 網路請求監控工具
- 手機模擬器或實體設備
- API 測試工具（如 Postman）

## 注意事項

1. **測試環境一致性**：確保測試環境與生產環境一致
2. **數據準備**：準備足夠的測試數據
3. **權限測試**：確保不同角色的權限測試完整
4. **邊界測試**：測試極端情況和邊界條件
5. **回歸測試**：修復問題後進行回歸測試
6. **文檔記錄**：詳細記錄測試過程和結果
